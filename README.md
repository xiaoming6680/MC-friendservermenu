# FriendServerMenu

FriendServerMenu 是一个面向朋友服的轻量级 Fabric MOD。玩家可以用 `/menu` 打开真正的客户端自定义“小铭的服务器菜单”，进行公共地点传送、坐标分享、活动通知、状态查看，以及 OP 服主管理操作。

## 支持版本

- Minecraft Java Edition `1.21.11`
- Fabric Loader `0.19.2` 或更新版本
- Fabric API `0.141.4+1.21.11`
- Java 21

## 构建

```powershell
.\gradlew.bat build
```

构建完成后，MOD jar 会出现在 `build/libs/`。

## 安装到服务端

1. 安装 Minecraft `1.21.11` 对应的 Fabric 服务端。
2. 将 Fabric API 放入服务端 `mods` 文件夹。
3. 将构建出的 `friendservermenu-1.0.0.jar` 放入服务端 `mods` 文件夹。
4. 启动服务端，MOD 会自动生成 `config/friendservermenu.json`。

## 安装到客户端

1. 安装 Minecraft `1.21.11` 对应的 Fabric 客户端。
2. 将 Fabric API 放入客户端 `mods` 文件夹。
3. 将同一个 `friendservermenu-1.0.0.jar` 放入客户端 `mods` 文件夹。

客户端和服务端都需要安装此 MOD。客户端负责绘制自定义 Screen，服务端负责真实操作和权限检查。

## 配置公共地点

首次启动后会生成：

```text
config/friendservermenu.json
```

示例：

```json
{
  "menuTitle": "小铭的服务器菜单",
  "locations": []
}
```

不会自动生成示例传送点。玩家可以在传送页点击“新增传送点”，提交后由服务端校验并写入 `locations`。普通玩家新增时不需要填写 ID，服务端会按名称自动生成不重复 ID；中文名称会生成 `tp_<hash>` 形式的 ID，避免退化成固定 ID。新增失败时，错误会显示在提交按钮旁；新增成功后会返回传送页。普通玩家可以编辑已保存传送点，但不能修改传送点 ID；只有 OP 可以删除传送点。修改配置后，OP 可以在菜单里点击“重载配置”，GUI 会在实时刷新中看到最新传送点。

## 命令

- `/menu`：打开小铭的服务器菜单。
- `/adminmenu`：直接打开服主管理分页，仅 OP 等级 2 及以上可用。

## 普通玩家功能

- 查看服务器配置中的公共地点。
- 点击地点卡片后请求服务端传送。
- 在传送页新增或编辑公共传送点，维度通过下拉菜单选择；`使用当前位置` 会填入坐标并显示提示，X/Y/Z 输入框旁提供 `-1` / `+1` 微调按钮。
- 查看并复制当前客户端坐标。
- 请求服务端公开广播真实坐标，聊天消息会附带绿色 `[点我传送]` 按钮。
- 请求服务端私发真实坐标给自己。
- 坐标页的“复制坐标 / 公开坐标 / 私发坐标”是紧凑按钮，点击后会自动关闭 GUI。
- 传送、活动传送、活动通知和服主管理等一次性动作点击后会自动关闭 GUI；右上角 `×` 可直接退出 GUI。
- 查看在线人数、维度、坐标、时间、天气、MSPT/TPS。

## OP 等级 2 及以上功能

- 删除公共传送点；编辑功能普通玩家也可用，但服务端会阻止普通玩家改 ID。
- 时间管理：设置白天、正午、夜晚、午夜。
- 天气管理：设置晴天、下雨、雷暴。
- 玩家管理：将所有在线玩家传送到自己当前位置、恢复全员生命、补满全员饥饿值。
- 在线玩家管理：玩家列表只显示玩家名，点进单玩家二级页后可切换生存/创造/冒险/旁观模式，传送到腐竹或腐竹传送过去，暂时授予 10 分钟飞行或撤销飞行，恢复生命、补饥饿、清除状态效果、熄灭、Kick、Ban，并可选择授予 OP 1-4 级或撤销 OP。
- 清理管理：清理掉落物、经验球、箭矢类投射物、敌对生物。
- 重载 `friendservermenu.json`。
- 在活动页选择模板分类：集合活动、发物品、维护通知、探索/副本；不需要集合点的模板不会显示集合地点和传送开关。
- 发物品模板会由服务端校验物品 ID 和数量；玩家需要在聊天栏点击绿色 `[手动领取]`，或在活动 GUI 点击“领取物品”，服务端会记录每名玩家的领取状态并防止重复领取。
- 维护模板支持维护时间和维护倒计时；倒计时会在活动页实时显示，服务端会在 60/30/15/10/5/4/3/2/1 秒提醒并到点执行 `/stop`。
- 活动发出后，OP 可在活动页用“提前结束活动”或“结束活动”按钮关闭当前活动；带集合点的活动聊天按钮会打开 `/menu activity`。活动发布时离线的玩家加入服务器后会收到一次活动聊天提示，不会反复刷屏。

普通玩家不能组织活动。普通玩家即使伪造客户端 payload 请求管理员 action 或活动组织 action，服务端也会拒绝并发送提示消息。

## GUI 与同步

- GUI 会按约每 1 秒向服务端请求最新菜单数据和服务器状态。
- 公共传送点列表、在线人数、维度、坐标、24H 制世界时间、天气、MSPT/TPS 会随刷新更新。
- 页面控件按当前位置即时绘制，避免输入框或按钮背景覆盖页面文字。
- 左侧导航栏和右侧内容区都有独立滚轮，窗口较小时不会把按钮挤出屏幕。
- 坐标页的复制、公开、私发按钮在鼠标悬停时会在按钮组下方显示作用说明。
- 所有自定义按钮统一播放 `friendservermenu:gui_click`，资源映射在 `assets/friendservermenu/sounds.json`，当前使用 Vorbis 编码的 `assets/friendservermenu/sounds/button.ogg`；客户端也会兜底注册该 SoundEvent。

## 坐标传送按钮

公开坐标时，服务端会记录一个 5 分钟有效的临时坐标分享 ID，聊天栏绿色 `[点我传送]` 按钮只执行 `/fsm_coord_tp <id>`。传送目标坐标和维度由服务端保存，玩家客户端不会提交可信坐标。

## 不包含的功能

本 MOD 不实现领地、经济、商店、权限组、大厅、反作弊、数据库或网页后台。
